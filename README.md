## üìå Project Overview
A one-stop portal designed for students to track their wellness activities, access mental health resources, and manage a healthy lifestyle throughout their academic journey.

This project was developed to help students balance academic demands with personal well-being. It features a secure authentication system, a registration module, and an interactive dashboard.

## üöÄ Key Features
- **Authentication System**: Login and account registration using `localStorage` for data persistence.
- **Wellness Dashboard**: A centralized view of health activity summaries.
- **Activity Management**: Categorized lists for physical, mental, and academic activities.
- **Resource Library**: Access to curated health articles (Physical, Mental, and Lifestyle).
- **Dark Mode**: A toggleable display option for better user experience and eye comfort.
